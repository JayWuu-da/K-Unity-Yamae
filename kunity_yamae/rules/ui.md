# Unity UI Rule Card

- Inspect Canvas, CanvasScaler, GraphicRaycaster, EventSystem, selected input module, active hierarchy, interactable state, CanvasGroup, raycastTarget, RectTransform size, sorting order, and persistent onClick bindings.
- Prefer controller, presenter, or runtime binding fixes over direct prefab YAML edits.
- If a persistent listener, prefab reference, layout, or raycast path may be broken, report a manual Unity Editor check.
